Sultan Al Isaee ~ foxoman @2022

Mersal is an SMS/OTP API Wrapper for `textbelt` (`http://textbelt.com`) public API,
that is built for developers who just want to send and receive SMS in nim.

 ### install
 ```nimble install mersal```
